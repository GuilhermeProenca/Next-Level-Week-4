## Required! Install git, node and yarn (or npm)
```
# Clone Repository
$ git clone https://github.com/GuilhermeProenca/Next-Level-Week-4.git

# Go to server folder
$ cd Next-Level-Week-4

# Install Dependencies
$ yarn

# Run Aplication
$ yarn dev

# Access localhost
http://localhost:3000
```
